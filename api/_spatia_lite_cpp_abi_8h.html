§ SPATIALITECPP_ABI

#define SPATIALITECPP_ABI

Defines the library ABI for Windows otherwise it is blank.

The following combinations of ABIs are supported

  • SPATIALITECPP_EXPORT is defined then a dynamic library is being built and the DLL export spec is defined.
  • SPATIALITECPP_IMPORT is defined then a dynamic library is being used and the DLL import spec is defined.
  • Neither SPATIALITECPP_IMPORT nor SPATIALITECPP_EXPORT are defined then a static library is being built or used.
