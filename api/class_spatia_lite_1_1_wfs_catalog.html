Loads data from some WFS source (using WFS paging)

Parameters
[in]databaseCurrent spatial database
[in]pathPointer to some WFS-GetFeature XML Document (could be a pathname or an URL).
[in]uriAn alternative URI for DescribeFeatureType to be used if no one is found within the XML document returned by GetFeature.
[in]layerThe name of the WFS layer.
[in]swapIf TRUE the X and Y axes will be swapped
[in]tableThe name of the table to be created
[in]primaryName of the Primary Key column; if NULL or mismatching then "PK_UID" will be assumed by default.
[in]spatialIf TRUE an R*Tree Spatial Index will be created
[in]pageMax number of features for each single WFS call; if zero or negative a single monolithic page is assumed (i.e. paging will not be applied).
[in]rowsOn completion will contain the total number of actually imported rows
[in]progressPointer to a callback function to be invoked immediately after processing each WFS page (could be NULL)
[in]callbackAn arbitrary pointer (to be passed as the second argument by the callback function).
